# Requirements

The new functionality that the business analysts want
consists of:

- [x] The web application must show a home page when
      they enter to the root URL of the current web site.
- [ ] The web application must allow the user to add a new
      field into bank accounts data: country. The users must
      be able to submit the account country when they are
      creating a new account

In addition to this, your IT director asked you to address the technical debt that currently exists on the project:

- [ ] Write the Software Requirements Specification and
      Software Design Document documentation.
- [ ] Write Unit tests and Functional tests for the backend
      API code.
- [ ] Set GitHub flow as the branch-based workflow strategy
- [ ] Use Azure Application Insights to enable monitoring of
      the application
- [ ] Deploy the User Acceptance Testing (UAT)
      environment. This environment will mimic the
      Production Environment and will be used to test the
      application before releasing to production.
